# Connect Four
![コネクトフォー](https://www.nintendo.co.jp/switch/as7ta/games/img/games/connectfour.jpg)
(画像は任天堂公式サイトから引用)  

## 概要
- Nintendo Switch 『世界のアソビ大全51』に収録されている『コネクトフォー』をJavaで再現します。

## ルール
- コマを落として、先に縦または横または斜めに4つ並べたプレイヤーが勝ち。
- 重力つき四目並べとも言う。
